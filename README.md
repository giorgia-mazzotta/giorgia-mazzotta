### 1. Master's Thesis
* **Title:** Numerical Analysis of the Ignition Transient of Solid Rocket Motors
Focus on the P80 VEGA launcher SRM using Computational Fluid Dynamics (CFD) and Quasi-1D unsteady numerical model SPIT (Solid Propellant rocket Ignition Transient)

### 2. Solid Rocket Motors Courseworks
  * **Optimal Number of Stages:** static and dynamic structural coefficient analysis + gravitational losses model analysis
  * **Staging Design Principles:** Optimal Staging Configuration of the Minotaur IV+
  * **0D Internal Ballistics (P80 SRM):** quasi-steady state; energy equation and non-ideal parameters
  * **1D Internal Ballistics (P80 Mock-up):** variation of properties along the grain and nozzle length with the effect of increasing the volumetric loading fraction (changing J parameter)
  * **Ignition Transient Analysis (MOSPIT):** Conducted parametric simulations using the MOcking Solid Propellant rocket Ignition Transient code to assess the influence of pressurizing gas, mass flow rate variations, seal rupture, and grain geometry.

### 3. Aerospace Thermal Structures Courseworks
* Sizing of HRSI tiles during critical thermal conditions: analytical steady-state model, 1D numerical validation, 3D transient simulation
* Analysis of the thermal insulation performance of the air gap of a cabin window
* Thermoelastic analysis of a bolt of a mechanically fastened joint subjected to a linear thermal gradient
